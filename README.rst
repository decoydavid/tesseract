Tesseract
=========

An interactive cube art project.

Hardware
========

This project has, in the past, relied on a Raspberry Pi and the TLC5940 16 channel LED driver.


Physical dimensions
===================

Cube made from 2m wooden beams on four wooden feet of ~0.3m.
